<p>Это приложение для создания и управления товарами и услугами в магазине (в админ части), а также для покупок товаров и услуг в магазине.</p>
<p>Это проект на Laravel, который использует Inertia.js для создания приложения с высокой производительностью и удобным разработкой.</p>
Админка разаработана на laravel/blade, магазин - на Inertia/React.
Стек технологий:</p>
<ul>
    <li>PHP 8.5</li>
    <li>Laravel 12</li>
    <li>Inertia.js</li>
    <li>Tailwind CSS</li>
    <li>Vite</li>
    <li>RabbitMQ</li>
    <li>S3</li>
    <li>Redis</li>
    <li>Jenkins</li>
    <li>Telescope</li>
    <li>MySQL</li>
    <li>Docker</li>
</ul>
<p>Точка входа на сайт: http://localhost:8000/catalog</p>
